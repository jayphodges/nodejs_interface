### Ruby Goldberg Machine Interface
 A Node.js frontend for a Ruby API backend for retrieving names from the census.

### Prerequisites to Run Locally

  * [Node.js](https://nodejs.org/)

### Running Locally

  * Clone this repo.
  * ```npm install```
  * ```npm run build```
  * ```npm start```
